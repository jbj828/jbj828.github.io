Install NodeJS on Ubuntu Server

Mar 06, 2020 in AWS

Ubuntu Server에 NodeJS 실습환경 구축

NodeJS 설치

설치

참고 : NodeSource

PPA를 통하여 최신 버전을 가져온다

curl -sL https://deb.nodesource.com/setup_13.x | sudo -E bash -

우분투에 NodeJS를 설치
sudo apt-get install -y nodejs

NodeJS와 NPM이 잘 깔렸는지 확인
node -v
npm -v

NPM이 제 기능을 하게 하기 위해 다음 명령어 실행(이거 없으면 npm install 시 에러 날 확률이 높다)
sudo apt-get install build-essential

NodeJS Application 생성

cd /{디렉토리 이름a}
ls -al

sudo mkdir {디렉토리 이름b}
cd {디렉토리 이름b}

sudo chown ubuntu . : 오너를 ubuntu로 바꾼다


디렉토리에 웹 애플리케이션 생성

npm init
ls -al

ExpressJS 프레임웍 설치

npm install express
ls -al
ls -al node_modules/

웹 애플리케이션 구동

vi app.js

sudo node app.js

{IP번호}:80 을 브라우저 주소에 입력
